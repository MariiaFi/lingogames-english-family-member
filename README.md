# Family Members — Learn & Practice

Мини-игра для изучения английской лексики на тему семьи.

## Особенности
- 17 слов на тему семьи (mother, father, parents, sister, brother, и т.д.)
- Игровой формат обучения
- Мгновенная обратная связь при ошибках
- Примеры предложений для каждого слова
- Адаптивный дизайн для мобильных устройств
- Не требует подключения к интернету

## Как использовать
1. Откройте `index.html` в браузере
2. Нажмите "Начать игру"
3. Выбирайте правильный перевод английских слов
4. Получайте объяснения после каждого ответа
5. Следите за своим прогрессом

## Технические детали
- Чистый HTML, CSS и JavaScript
- Без фреймворков и зависимостей
- Подходит для размещения на GitHub Pages

## Словарь
Игра включает следующие слова:
- mother, father, parents
- sister, brother, siblings
- son, daughter, children
- husband, wife
- grandparents, grandmother, grandfather
- aunt, uncle, cousin
